博客搭建记录

发表于 2017-09-25

近些日子没事做（主要是懒得去找工作），不小心在网上看到了一篇使用github+hexo简单搭建博客的教程，又考虑到自己平时喜欢收集些图片，就想着做一个收录自己喜欢图片的网站来时不时看一下安慰一下自己，哈哈哈哈。
话不多说，下面开始教程（其实我看得别人教程，自己不断踩坑摸索出来的）。
大致可以分为如下几步：
1、搭建环境准备
2、安装Hexo
3、配置Hexo
4、将Hexo与Github page联系起来
5、安装主题
6、发布文章
7、使用MarkDown编写文章
8、站点配置文件以及主题配置文件

搭建环境准备

Node.js的安装和准备

下载Node.js安装文件
Node.js 64位 Node.js 32位
正常默认安装即可
安装完成之后，打开CMD命令行界面，输入
node -v
npm -v
看到下面界面就是安装成功了

Git的安装和配置

下载Git安装文件
Git 64位 Git 32位
保持默认安装即可
安装完成之后，打开CMD命令行界面，输入
git --version
出现下图结果，表示安装成功

接下来是github账户的注册和配置
打开网站https://github.com/
看到如下的注册界面

正常输入邮箱密码等，之后验证邮箱就可以激活
之后进入到Github界面创建代码库，如图所示

进入之后，创建代码库名字，命名为yourname.github.io（yourname为你想设置的名字）
参考如下图

点击创建即可
接下来你会看到如下界面

点击settings（你的code界面下可能只有一个README.md，属于正常现象）下拉到GitHub Pages目录下
在Source下选择master branch之后等待15分钟就可以访问https://yourname.github.io

ps:由于我的绑定了自定义域名，所以会显示
Your site is published at http://wokamo.info/
你的会显示
Your site is published at http://yourname.github.io/

安装Hexo

在一个自己喜欢的位置创建Hexo文件夹
例如我在D盘下创建D:/Hexo
之后进入到你创建的Hexo文件夹，右键可以看到Git Bash here，点击会出现如下界面

在命令行输入
npm install hexo-cli -g
之后会看到如下界面
未完待续
再输入
npm install hexo --save
在经过一系列白字之后，在命令行输入
hexo -v
若出现如下界面，表示安装成功

配置Hexo

初始化Hexo
在这里，我在D盘下创建wokamo文件夹，然后进入wokamo文件夹，右键Git Bash here，输入
hexo init
之后输入
npm install
等待一大堆白字的相伴之后，我们可以体验首次安装好Hexo的效果了
在命令行继续输入
hexo g
该命令为部署hexo，经历一大堆info后，输入
hexo s
然后会提示

接着在浏览器输入https://localhost:4000/将会看到

接下里将会使Hexo和Github pages联系起来

Hexo和Github pages联系起来

配置Git个人信息

进入到D:/wokamo，右键Git Bash here，在命令行中输入
设置Git的username和email
git config --global user.name "username"
git config --global user.email "XXX@XXX.com"
未完待续

配置_config.yml文件

打开D:/wokamo，找到_config.yml文件，记事本打开后，找到deploy:项，更改文件信息
deploy:
type: git
repo: git@github.com:yourname/yourname.github.io.git
branch: master
到此配置结束
